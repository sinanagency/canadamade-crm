Developer Instructions

CanadaMade Gulf Expo — Lead Capture App
Scope: Set up email and SMS notifications for the Gulf Expo lead capture app. You have full access to the domain, Supabase, and Netlify. WhatsApp integration will be handled separately.

Task 1: Email Setup (SendGrid)

Account Setup

  1. Create a SendGrid account at sendgrid.com (free tier: 100 emails/day)
  2. Go to Settings → Sender Authentication → Domain Authentication
  3. Enter domain: canadamade.com
  4. SendGrid will generate 3 CNAME DNS records
  5. Add the DNS records to the domain and verify

API Key

  1. Go to Settings → API Keys → Create API Key
  2. Name: CanadaMade Expo
  3. Select Restricted Access → enable Mail Send → Full Access
  4. Copy the API key (shown only once)
  5. Add to Netlify: Site Settings → Environment Variables
  6. Variable name: SENDGRID_API_KEY

Create Function

Create netlify/functions/send-email.js

  • Parameters: recipient email, first_name, flavor
  • Send from: info@canadamade.com
  • Fetch template from Supabase message_templates where name = customer_confirmation_email
  • Replace {{first_name}} and {{flavor}}
  • Send via SendGrid API
Provide to Me
  • SendGrid login credentials (email + password)
  • Confirmation that API key is added to Netlify

Task 2: SMS Setup (Unifonic)

Account Setup

  1. Create account at unifonic.com
  2. Complete business verification
  3. Go to Dashboard → API Settings
  4. Copy your App ID
  5. Add to Netlify environment variables:
    • UNIFONIC_APP_ID = your App ID
    • UNIFONIC_SENDER_ID = CANADAMADE (placeholder)

Create Function

Create netlify/functions/send-sms.js

  • Parameters: recipient phone, first_name, flavor
  • Fetch template from Supabase message_templates where name = customer_confirmation_sms
  • Replace {{first_name}} and {{flavor}}
  • Send via Unifonic API

Testing

Test the function with your own phone number before handoff.

Provide to Me
  • Unifonic login credentials (email + password)
  • Confirmation that environment variables are added to Netlify

Task 3: Connect to App Flow

Integration

After a customer completes verification, trigger notifications based on their preference:

comm_preference Action
email Call send-email with email, first_name, flavor
sms Call send-sms with phone, first_name, flavor
whatsapp Skip (will be handled separately)

Error Handling

  • Log success/failure to console
  • Don't block user flow if notification fails

Task 4: Staff Notification (Prepare Only)

Create netlify/functions/notify-staff.js but don't connect yet.

Requirements

  • Accept full customer data
  • Fetch template from message_templates where name = staff_notification
  • Replace variables: {{first_name}}, {{last_name}}, {{company}}, {{job_title}}, {{phone}}, {{email}}, {{country}}, {{flavor}}, {{photo_url}}, {{timestamp}}

I will provide staff phone number and delivery method later.
